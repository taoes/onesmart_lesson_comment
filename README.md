# 精锐评课辅助插件

### 安装步骤

+ 下载安装包 `https://github.com/taoes/onesmart_lesson_comment/archive/master.zip`
+ 解压安装包到指定位置，如D盘

+ 在Chrome浏览器地址栏目输入 ： `chrome://extensions/`
+ 开启右上角的开发者模式
+ 点击左侧 `加载已解压的拓展程序`即可
